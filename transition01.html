트랜지션(Transition) : 화면전환 A -> B, hover, checked시 활용

transition-duration : 실행시간

transition-duration:0.5s; /* => 500ms; */


transition-delay : 지연시간

transition-delay:0.2s; /* => 200ms; */


transition-property : 트랜지션을 적용할 css 속성을 지정

transition-property:width, height; /* css의 속성을 지정 */
transition-duration:0.5s, 0.9s;

transition-timing-function : 시간함수를 적용하여 가속 또는 감속효과

transition-timing-function:ease-in | ease-out | ease-in-out | ease | linear | cubic-bezier() /* cubic-bezier(.94,-0.65,.35,1.63) */

애니메이션(Animation) : 움직이는 그림 - 연속동작

animation은 @keyframes 규칙에 의해 그 동작이 정의됨

animation-name : 애니메이션명으로 keyframes규칙에 해당 애니메이션명을 기재해야하며, 적용하고자 하는 요소에도 같이 기재가 되어야 함.

animation-duration : 애니메이션이 적용될 시간을 기재

.box { animation-name:ani1; animation-duration:10s; }

@keyframes ani1 {
from { left:100px; } /* 0초 */
30% { left:1500px; } /* 3초기점 */
60% { left:200px; } /* 6초 기점 */ to { left:800px; } /* 10초 기점 */
}



animation-delay : 애니메이션 지연시간 지정

animation-timing-function : 애니메이션의 가감속 효과 지정

animation-iteration-count : 애니메이션의 반복 실행 횟수(무한 : infinite)

animation-direction : 애니메이션의 진행방향(normal, reverse, alternate, alternate-reverse)
normal : 정방향 애니메이션

reverse : 역방향 애니메이션

alternate : 정-역방향 애니메이션

alternate-reverse : 역-정방향 애니메이션

animation-fill-mode : 기본 애니메이션은 애니메이션을 실행한 후 원래대로 복귀되는 속성이 있어 종료된 to에서 멈춰있게 하고 싶을 때 사용
none, forwards,backwards : 원래대로 돌아감

both : 원래대로 돌아가지 않고, 종료된 상태로 정지됨

animation-play-state : 애니메이션을 특정 시점에서 정지시키거나 실행을 제어하는 속성
paused : 일시정지

running : 일시정지 후 계속실행
